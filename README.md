# Sito Laboratorio ABC

Template statico per il sito di un laboratorio, pronto per essere pubblicato con GitHub Pages.

## Contenuto del progetto

- `index.html` - homepage
- `team.html` - membri del laboratorio
- `progetti.html` - progetti e collaborazioni
- `pubblicazioni.html` - articoli e lavori scientifici
- `contatti.html` - recapiti e sede
- `style.css` - stile del sito
- `script.js` - menu mobile

## Come pubblicarlo su GitHub Pages

1. Crea un nuovo repository su GitHub.
2. Carica tutti i file di questa cartella nel repository.
3. Vai su **Settings > Pages**.
4. In **Build and deployment**, seleziona:
   - **Source**: Deploy from a branch
   - **Branch**: `main` e cartella `/root`
5. Salva.
6. Dopo la pubblicazione, il sito sarà disponibile all'indirizzo fornito da GitHub Pages.

## Personalizzazioni consigliate

- Sostituire "Laboratorio ABC" con il nome reale del laboratorio
- Aggiornare email, indirizzo, link GitHub e contenuti delle pagine
- Aggiungere logo e immagini nella cartella `images/`
- Collegare un dominio personalizzato se necessario

## Suggerimento

Per un sito accademico più facile da aggiornare nel tempo, in una fase successiva si può passare a Jekyll o Astro mantenendo la stessa struttura dei contenuti.
